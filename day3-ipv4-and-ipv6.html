IPV4 and IPV6.

IPV4 vs IPV6: A brief comparision

IPV4 is the fourth iteration of internet protocol which powers most of internet traffic.It uses the 32 bit addresses and it is represented as four decimal numbers seperated by the dots. example network adress (192.168.1.0). Key features of the IPV4 is:- adress format:- it can have total of 4.3 billion possible adresses. it uses subnet masks to divide the networks..

The limitations of the IPV4 are:-

The address will be exhausted due to limited 32-bit space. and also it relies on workarounds like network adress translation to conserve the addresses.

IPV6:-Future of THe internet.

IPV6 is the modern protocol which uses the 128-bit addresses and it is written in eight hexadecimal seperated by colons.example (2001:0db8:85a3::8a2e:0370:7334)

Key features of the IPV6 are:

It has the vast adress space which is almost 340 undecillion which is enough for every device on earth and beyond.

It simplifies routing by using hierarchial addressing which will reduce the routing table sizes.

It has built in security IPsec which is used for encryption and authentication.

there is no need for network adress translation here.

There is no adress exhaustion concerns and it has improved performance and also auto configuration and also has better support for IOT and future technologies.
