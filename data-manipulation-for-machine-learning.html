Machine learning is the study of algorithms that can learn from experience.

In machine learning, what we do is we define a flexible program whose behavior is determined by a number of parameters. We then use a dataset to determine the best possible parameters, and using these parameters we can manipulate the behavior of the program.

If the parameters are fixed, then we call the program a model.

The set of all distinct programs that we can produce just by manipulating the parameters is called a family of models.

What is a Tensor?

A Tensor represents an array of numerical values. It can be multi-dimensional:

  • 1D → Vector
  • 2D → Matrix

  • 2D → Tensor

arange() Function

It creates a vector of evenly spaced values starting at 0 (included) and ending at n (not included), i.e., 0 <= f(x) < n. We can also set the interval size. If we don't specify it, the default is 1. These are stored in main memory and meant for CPU-based computation.

Each value is called an element of the tensor.

Example:

import torch 
x = torch.arange(12)
x

This creates a vector holding 12 values, each with a default difference of 1.

To find the number of elements:

x.numel()

To check the shape:

x.shape

To reshape:

X = x.reshape(3, 4)
X

Now it's a 3x4 matrix.

Zeros and Ones

Often we need arrays of zeros or ones:

torch.zeros(12)  # 12 zeros

Reshape using:

torch.zeros(2, 3, 4)  # 2 tensors of shape 3x4

Same applies for:

torch.ones(2, 3, 4)

Random Values with randn

Instead of manually inputting values, use:

torch.randn(3, 4)

This gives a 3x4 matrix with random values, generally centered around 0.

Indexing and Slicing

Very similar to Python lists:

x[-1]         # Last row
x[1:3]        # 2nd and 3rd rows
x[1, 2] = 13  # Change value at 1st row, 2nd col to 13
x[:2, :] = 12 # Set first two rows to 12

Exponential and Arithmetic Operations

torch.exp(x)

You can also perform addition, subtraction, multiplication, division, etc.

Concatenation

Combine two tensors:

torch.cat((x, y), dim=1)

For two [3, 4] tensors, this results in a [3, 8] tensor.

Check for equality element-wise:

X == Y

Broadcasting

If two tensors have shapes [3,1] and [1,2], broadcasting allows you to compute a [3,2] result.

In-place Operations

before = id(Y)
Y = X + Y
before == id(Y)  # False

To modify in place:

Y[:] = X + Y

Another way:

z = torch.zeros_like(Y)
print(id(z))
z[:] = x + y
print(id(z))  # Same as before

Conversion Between Tensors and NumPy

Convert tensor to NumPy array:

A = x.numpy()

Convert NumPy to tensor:

b = torch.from_numpy(A)

You can also use built-in Python functions like:

float(), int(), item()
