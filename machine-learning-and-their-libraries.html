Understanding Data Handling, Linear Algebra, and Machine Learning Basics in Python

Understanding Data Handling, Linear Algebra, and Machine Learning Basics in Python

CSV and Pandas Basics

A CSV (Comma Separated Values) file stores tabular data where each value is separated by a comma.

To read CSV files easily in Python, we use the pandas library:

import pandas as pd

data = pd.read_csv('data_file.csv')
print(data)

This prints all the values present in the CSV file.

Handling Missing Values in Data

Sometimes, datasets have missing or null values. There are two ways to handle them:

  • Imputation: Replace missing values with estimates.
  • Deletion: Remove rows or columns containing missing values.

A common approach is to fill missing values with the mean of that column.

In pandas, the function to do this is fillna():

inputs = inputs.fillna(inputs.mean())

This fills all missing data with the column mean.

From Data to Tensors: Linear Algebra Basics with PyTorch

After cleaning data, we convert it to tensor format for numerical operations.

Dot Products

Dot product is fundamental in linear algebra.

In PyTorch, you find it like this:

torch.dot(x, y)

Matrix-Vector Products

To multiply a matrix with a vector, use:

torch.mv(A, x)

Or use the @ operator which can also do cross products.

Matrix-Matrix Multiplication

Use:

torch.mm(A, B)

Or simply:

A @ B

Norms of Vectors

The norm measures the size or length of a vector.

Properties of norms include:

  1. When you multiply a vector by a scalar, its norm is multiplied by the absolute value of that scalar.
  2. The triangle inequality holds, meaning the norm of the sum of two vectors is less than or equal to the sum of their norms.
  3. The norm is always non-negative and is zero only if the vector itself is zero.

The L2 norm (also called Euclidean norm) of a vector is the square root of the sum of the squares of its components.

In PyTorch, you can calculate this using:

torch.norm(x)

The L1 norm (Manhattan norm) is the sum of the absolute values of the vector components:

torch.abs(x).sum()

Visualizing Data with Matplotlib

To visualize data, matplotlib is useful. The set_axes() function lets you set labels, ranges, and scales for plots.

Linear Neural Networks for Regression

Let’s start simple with linear regression.

We assume the relationship between features and the target is linear.

For example, the predicted price is calculated by multiplying the weight assigned to the area by the area value, plus the weight assigned to the age by the age value, plus a bias term.

More generally, if there are many input features, the prediction is the sum of each feature multiplied by its corresponding weight, plus the bias.

We use a loss function to measure prediction error, often the squared difference between predicted and actual values for regression tasks.

Gradient Descent

Gradient descent is an optimization method that iteratively reduces loss by adjusting parameters in the direction that decreases error.

At each iteration, we randomly select a small batch of data, calculate the gradient of the loss with respect to parameters on this batch, multiply it by a learning rate (a small positive number), and update the parameters accordingly.

Normal Distribution Visualization

To calculate the value of a normal distribution (Gaussian) for a given input, mean, and standard deviation, use this function:

import math
import numpy as np

def normal(x, mu, sigma):
    coefficient = 1 divided by the square root of (2 times pi times sigma squared)
    exponent = negative one-half times (x minus mu) squared divided by sigma squared
    return coefficient times exponential of exponent

This function returns the probability density of the normal distribution at x.

NumPy: Numerical Python

Features:

  1. The main data structure is ndarray which supports efficient multi-dimensional arrays.
  2. Provides functions for linear algebra, statistics, and random number generation.
  3. Supports element-wise operations without explicit reshaping.
  4. Works well with other libraries like pandas and matplotlib.
  5. Allows faster and memory-efficient computations.

How to Use NumPy

Install:

pip install numpy

Import:

import numpy as np

Create array from list:

list1 = [10, 20, 30]
arr1 = np.array(list1)
print(type(arr1))  # numpy.ndarray

Common functions:

  • np.arange(start, stop, step) creates evenly spaced values.
  • np.zeros(shape) creates an array of zeros.
  • np.ones(shape) creates an array of ones.
  • np.linspace(start, stop, num_steps) creates evenly spaced numbers over an interval.
  • np.eye(n) creates an identity matrix.
  • np.random.rand(shape) creates samples from a uniform distribution between 0 and 1.
  • np.random.randn(shape) creates samples from the standard normal distribution.
  • np.random.randint(low, high) generates random integers between low and high.
  • np.random.seed() sets the random seed for reproducibility.
  • reshape() changes the shape of arrays.

Helper functions include:

  • min(), max(), argmin(), argmax()
  • shape gives the dimensions of the array.
  • dtype shows the data type of array elements.

You can slice arrays and assign values like this:

arr[0:4] = 100

Conditional selection example:

arr = np.arange(1, 11)
print(arr[arr > 2])  # prints values greater than 2

Mathematical functions supported:

  • square root, exponential, sine, logarithm, mean, etc.

Axis-wise operations are also possible for multi-dimensional arrays.

Pandas for Data Manipulation

Pandas helps with:

  • Reading and writing Excel, SQL, and CSV files.
  • Cleaning and preparing data.
  • Reshaping data.
  • Visualizing data with built-in plotting.

A DataFrame is like a table with rows and columns.

Create DataFrame from array:

import pandas as pd

data = np.array([[1, 2, 3], [4, 5, 6]])
df = pd.DataFrame(data, columns=['A', 'B', 'C'])
print(df)

Output:

   A  B  C
0  1  2  3
1  4  5  6

Create DataFrame from dictionary:

dict = {'name': ["aparna", "pankaj", "sudhir", "Geeku"],
        'degree': ["MBA", "BCA", "M.Tech", "MBA"],
        'score': [90, 40, 80, 98]}

df = pd.DataFrame(dict)
print(df)

Output:

    name  degree  score
0  aparna     MBA     90
1  pankaj     BCA     40
2  sudhir  M.Tech     80
3   Geeku     MBA     98

Pandas Series

Create Series from dictionary:

data_dict = {'Pandas': 10, 'in': 20, 'python': 30}
ser = pd.Series(data_dict)
print(ser)

Output:

Pandas    10
in        20
python    30
dtype: int64

Useful Pandas Functions

  • head() — shows first 5 rows.
  • tail() — shows last 5 rows.
  • info() — provides summary information.
  • sort_values() — sort by column values.
  • sort_index() — sort by index.
  • reset_index() — reset the index.
  • rename() — rename columns or rows.
  • melt() — transform columns into rows.
  • drop() — remove rows or columns.
  • at() — access a single value by label.
  • filter() — filter rows or columns.
  • where() — apply conditional filtering.
  • query() — query data using expressions.

Combining DataFrames

  • merge() — perform joins like left, right, or outer join.
  • concat() — concatenate DataFrames either by rows or columns.

Aggregations & Null Checks

  • value_counts() — count occurrences of unique values.
  • sum(), cumsum() — sum and cumulative sum.
  • var(), median() — variance and median.
  • isnull() — detect missing values.
  • isnull().sum() — count missing values per column.
