Introduction to Matplotlib and Regression Basics

matplotlib.pyplot is a module from the Matplotlib library that provides a simple interface for creating plots and adding elements like lines, text, and images. It’s a go-to tool for data visualization in Python.

Installing Matplotlib

To install Matplotlib, use the following command:

pip install matplotlib

Using Pyplot

The first step in using pyplot is to import it:

import matplotlib.pyplot as plt

Next, we create data using lists or arrays and plot it using plt.plot(). Additional elements like titles, axis labels, and legends can be added using specific methods. Finally, the plot is displayed using plt.show().

Customizing the Plot Area

We can use plt.figure() to gain more control over the plotting area. The figsize parameter allows us to set the figure's dimensions:

fig = plt.figure(figsize=(6, 4))
ax = fig.add_subplot(1, 1, 1)

add_subplot() returns the axes object that we can further customize.

Working with Legends

Legends help describe the plot elements. A few useful attributes are:

  • shadow: Adds a shadow behind the legend. Default is None.
  • markerscale: Adjusts the size of legend markers relative to the plot.
  • numpoints: Specifies the number of marker points in the legend for 2D lines.
  • fontsize: Sets the font size of the legend text.
  • facecolor: Sets the background color of the legend.
  • edgecolor: Sets the border color of the legend box.

Line Plot

A line plot is used to compare two variables. We can set:

  • color: Color of the line.
  • linestyle: Solid, dashed, dotted, etc.
  • linewidth: Thickness of the line.
  • marker: Shape of the data points (circle, triangle, etc.).
  • label: Name to be used in the legend.

Example:

plt.plot(x, y, color='blue', linestyle='--', linewidth=2, marker='o', label='Line A')
plt.legend()

Scatter Plot

A scatter plot shows the relationship between two variables using individual data points. If the points form a clear pattern, the variables are likely related.

plt.scatter(x, y)

Parameters include:

  • c: Color of the markers.
  • s: Size of the markers.
  • marker: Shape (circle, triangle, etc.).
  • edgecolors: Color of the marker edges.

Bar Chart

Bar charts are used to compare different categories. The bar() function creates vertical bars, while barh() creates horizontal bars.

plt.bar(x, y, color='green', edgecolor='black', align='center', label='Group A')
plt.legend()

Parameters:

  • color: Fill color of the bars.
  • edgecolor: Outline color.
  • label: For legends.
  • align: Aligns bars with the ticks on the axis.

Pie Chart

Pie charts are useful for showing proportions among a small number of categories.

plt.pie(values, labels=labels, startangle=90, autopct='%1.1f%%', colors=colors)

Parameters:

  • labels: Names of the slices.
  • startangle: Starting angle for the first slice.
  • autopct: String format for displaying percentage.
  • colors: Slice colors.

Histogram

Histograms show the distribution of data across intervals or “bins.”

plt.hist(data, bins=10, color='skyblue', density=True)
  • bins: Number of divisions.
  • color: Bar color.
  • density: If set to True, the histogram is normalized.

3D Line Plot

For 3D visualizations, use the following setup:

from mpl_toolkits.mplot3d import Axes3D

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot(x, y, z)

Setting projection='3d' enables 3D plotting.

Working with Images

Matplotlib can also be used to work with images:

img = plt.imread('image.png')
plt.imshow(img)
plt.axis('off')
plt.show()

Basics of Linear and Logistic Regression

Linear Regression

Linear regression is a supervised learning algorithm that fits the best possible straight line to a set of data points. It assumes a linear relationship between input and output:

y = mx + c

The objective is to minimize the error between the actual and predicted values, often using the least squares method.

Evaluation Metrics

  • Mean Squared Error (MSE): Average of squared differences between actual and predicted values.
  • Mean Absolute Error (MAE): Average of absolute differences.
  • Root Mean Squared Error (RMSE): Square root of MSE.

Logistic Regression

Logistic regression is a classification algorithm used in supervised learning. It transforms the continuous output of a linear function into a probability using the sigmoid function. The output is then mapped to a class label based on a threshold (typically 0.5).

Used for tasks where the output is categorical, such as spam detection or binary classification problems.

Bayesian Linear Regression

Bayesian linear regression incorporates prior knowledge into the model using Bayes’ Theorem:

P(A|B) = [P(B|A) * P(A)] / P(B)

Where:

  • P(A|B): Posterior probability after observing data.
  • P(B|A): Likelihood of the data given the parameters.
  • P(A): Prior belief about the parameters.
  • P(B): Marginal likelihood of the data.
